# Image asset guide

Temporary SVG image assets are organized by purpose so the pull request does not include unsupported binary files:

- `logo/bear-lake-jason-mark.svg`: temporary site mark used for the header and favicon.
- `placeholders/bear-lake-hero.svg`: temporary Bear Lake hero, supporting scenery, and social-sharing image.
- `placeholders/jason-headshot-placeholder.svg`: temporary Jason Petersen headshot image.

When binary uploads are supported, replace these with the supplied production logo, Bear Lake scenery, and Jason headshot, then update the paths in `src/data/site.ts`.
